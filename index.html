📖 Anleitung für neue Nutzer

Folge diesen Schritten um das System für dich einzurichten:

1Google Sheets Dateien kopieren

Öffne die Vorlagen und erstelle eine Kopie in deinem Google Drive (Datei → Kopie erstellen). Die Spreadsheet-ID findest du danach in der URL.

2Google Cloud OAuth Client-ID erstellen

1. console.cloud.google.com
2. Neues Projekt → Google Sheets API aktivieren
3. Anmeldedaten → OAuth-Client-ID (Webanwendung)
4. Autorisierte JavaScript-Quellen: deine GitHub Pages URL
5. Client-ID hier eintragen

3IDs eintragen & testen

Alle IDs oben eintragen und auf „Verbindung testen" klicken.

4Ersten Hund anlegen

🗂️ Stammdaten → 🐕 Hunde → Neuen Hund hinzufügen.

5App installieren (optional)

iPhone: Teilen → „Zum Home-Bildschirm"
Android: Menü → „Zum Startbildschirm hinzufügen"
